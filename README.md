# oracle-database-admin-scripts

Raccolta di **documentazione operativa** e **script** per la gestione di database Oracle (11g, 12c, 18c, 19c).

Il repository è pensato come raccolta di files per attività quotidiane di amministrazione, troubleshooting, performance tuning e procedure import-export (DataGuard, DataPump).

## Contenuti

- `Data-Guard/`
  - files utili per l'associazione DataGuard
- `monitoring/`
  - Script di monitoraggio (sessioni, locking, temp, undo, I/O)
  - Script di performance (AWR, ASH, SQL tuning, indici, statistiche)
  - Script di sicurezza (utenti, privilegi, audit, password policy)
- `troubleshooting/`
  - Script per problemi noti o ricorrenti con metodi di risoluzione
- `Import-procedure/`
  - Files utili per prcedure di export import in DataPump
- `manage-user/`
  - Comandi utili per la gestione degli utenti
